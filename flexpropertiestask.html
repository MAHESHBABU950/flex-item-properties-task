FLEX-BOX:

  • align content
    • centers the elements in y-axis
  • align items
    • it creates two equal boxes and center the elements in those boxes

    go through all the properties of these align elements

Flex-item properties:

  • order
    • it reorganizes the flex-items based on the order number
    • the highest order valued item always stays at right (ascending order)
  • flex-grow
    • when enables flex-item occupies the empty space in the flex-box
    • higher the flex-grow value, higher the priority
  • flex-shrink
    • flex-shrink:0
    • it stops the responsiveness (shrinking based on the screen size) of the flex-item
    • flex-shrink:200px (the item doesn't shrink less that 200px)
  • flex-basis
    • initial width of the flex-item
    • isn't much diffrent from the normal width of the flex item
  • flex
    • flex: flex-grow_value flex-shrink_value flex-basis_value
    • we can give all the mentioned properties to a flex-item in single command
